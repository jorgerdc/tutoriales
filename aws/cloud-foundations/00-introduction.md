# Introducción

## Tendencias en computación en la nube

### Principales novedades de AWS

* Generative AI. Este concepto está cambiando la forma en la que las empresas trabajan
* AWS define infraestructura, herramientas y aplicaciones para hacer uso de Generative AI
  en sus negocios.
* AWS Generative AI Services:
  * Amazon Bedrock: Servicio que permite a los desarrolladores crear y escalar aplicaciones.
    Permite la selección de high-performance foundation models (FMs)
  * Amazon Q: Servicio de IA generativa que permite a los desarrolladores crear aplicaciones
    de chatbots y asistentes virtuales.
    * Amazon Q Business Chat: Permite a los empleados interactuar con datos empresariales
      utilizando lenguaje natural.
    * Amazon Q Code: Permite a los desarrolladores generar, revisar y depurar código
      utilizando lenguaje natural.
